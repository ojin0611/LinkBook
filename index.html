    Linkbook이란?

    • PC, mobile 통합 링크 공유 어플리케이션
    • 간단한 동작만으로 링크 추가
    • 원하는 카테고리별 링크 분류

    Linkbook의 기능

    1. Facebook 또는 Google Account를 통해 로그인 가능
    2. PC : 버튼 또는 단축키 이용해 linkbook에 현재 링크 추가
    3. mobile : url을 복사하면 linkbook에 바로 추가할 수 있도록 알림 표시
    4. 검색을 통해 내 링크
    5. 드래그 & 드롭 으로 손쉬운 링크 추가
    6. 링크에 간단한 메모 덧붙이기 가능
    7. 관심분야에 대한 링크 추천
